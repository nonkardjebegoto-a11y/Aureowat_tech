---

## 🌍 À propos

**Aureowat Tech** est une startup tchadienne basée à Moundou, spécialisée en ingénierie technique du bâtiment et en solutions numériques. Notre cœur de métier couvre l'électricité bâtiment, le froid et la climatisation, ainsi que le développement web et la cybersécurité — des services adaptés aux réalités et besoins locaux du Tchad et de l'Afrique centrale.

---

## 🛠️ Services

| Service | Description |
|---|---|
| ⚡ **Électricité bâtiment** | Installation, câblage, tableaux électriques, mise aux normes |
| ❄️ **Froid & Climatisation** | Installation et maintenance de climatiseurs, chambres froides, groupes froid |
| 🌐 **Développement Web** | Sites vitrines, portails institutionnels, applications web sur mesure |
| 🔐 **Cybersécurité** | Audit de sécurité, tests de pénétration, sensibilisation |
| 📱 **Applications Mobile** | Applications React Native pour Android |
| 🎨 **UX/UI Design** | Interfaces modernes et accessibles |
